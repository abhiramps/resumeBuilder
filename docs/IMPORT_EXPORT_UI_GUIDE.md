## Tips for Users

1. **JSON Export**: Best for backup and data portability
2. **PDF Export**: Best for job applications
3. **DOCX Export**: Best for further editing
4. **Bulk Export**: Use for backing up all resumes
5. **Import**: Drag-and-drop is faster than browsing

## Common Scenarios

### Scenario 1: Backup All Resumes
1. Click "Select"
2. Click "Select All"
3. Click "Export Selected"
4. Save JSON file to cloud storage

### Scenario 2: Share Resume for Review
1. Click menu on resume
2. Select "Export"
3. Choose PDF
4. Email PDF to reviewer

### Scenario 3: Transfer Between Accounts
1. Export as JSON from Account A
2. Download file
3. Login to Account B
4. Import JSON file

### Scenario 4: Create Variations
1. Click menu on resume
2. Select "Duplicate"
3. Edit duplicate for different job
4. Keep original unchanged
